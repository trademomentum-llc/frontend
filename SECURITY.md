# Security Policy

We take the security of Trade Momentum LLC applications and services seriously. If you believe you have found a security vulnerability, please report it to us as described below.

## Supported Versions

Currently, we only support security updates for the active deployment branch and the latest releases:

| Version | Supported |
| ------- | --------- |
| Main (development/deployment) | :white_check_mark: Active Support |
| < v1.0.0 | :x: Not Supported |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a vulnerability, please report it by emailing us at [security@trademomentumllc.com](mailto:security@trademomentumllc.com). 

In your report, please include:
- A description of the vulnerability and its potential impact.
- Detailed step-by-step instructions (and if possible, proof-of-concept code or screenshots) to reproduce the vulnerability.
- Your contact information so we can coordinate the fix and disclosure.

## Our Commitment

If you report a vulnerability to us, we will:
- Acknowledge receipt of your report within 48 hours.
- Work to investigate and verify the vulnerability as quickly as possible.
- Provide an estimated timeline for a fix.
- Notify you once the vulnerability has been resolved.

We appreciate the responsible disclosure of security vulnerabilities and will work with you to ensure that the issue is resolved safely.
